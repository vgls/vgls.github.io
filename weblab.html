Labyrinthear's Mod Guide using FFZ and BTTV

Download FrankerFaceZ: https://www.frankerfacez.com/

Download BetterTTV: https://betterttv.com/



Ο συγκεκριμένος οδηγός περιλαμβάνει βασικές λειτουργίες και εργαλεία από τα δύο αυτά browser extension.
Μπορεί να γίνει ταυτόχρονη χρήση και των δύο αλλά το FFZ μπορεί αρκετά εύκολα να αντικαταστήσει το BTTV.
Για οποιαδήποτε απορία από αυτά που ακολουθούν μπορείτε να με κάνετε add στο Twitch για να μπορέσετε να μου στείλετε pm ή να μου στείλετε pm στο Discord: labyrinth#4987


Ξεκινώντας ως mod τα βασικά σου εργαλεία είναι το timeout, το delete και το ban.
όταν κάποιος γίνεται τα Mod Icons δεν εμφανίζονται από default, οπότε πρέπει να τα ενεργοποιήσουμε.
Παρακάτω φαίνεται ο τρόπος ενεργοποίησης.







Ένα από τα βασικά εργαλεία του BTTV αν όχι το βασικότερο είναι οι επιλογές για timeout που δίνει κάνοντας δεξί κλικ πάνω σε ένα username.
Εύκολα και γρήγορα με αυτό τον τρόπο έχουμε παραπάνω επιλογές.
Η συγκεκριμένη είναι default ρύθμιση και δεν χρειάζεται να την ενεργοποιήσουμε.







Άλλη μία default και αρκετά χρήσιμη λειτουργία του BTTV είναι το shortcut mention με double αριστερό click πάνω σε ένα username.
Με αυτό τον τρόπο αποφεύγουμε το auto completion που παρέχει το Twitch με το ''@'' όπου αρκετές φορές προκαλούνται λάθη.







Το BTTV έχει και κάποιες άλλες επιλογές στις ρυθμίσεις του όπου θα μπορούσαν να φανούν χρήσιμες σε κάποιους ανάλογα με τις προσωπικές του ανάγκες.
Πιο κάτω θα αναφερθώ και σε κάτι τελευταίο για το BTTV αλλά προς το παρόν θα αναφερθώ στο FFZ.
Το FFZ συγκριτικά με το BTTV παρέχει μια πιο εξειδικευμένη χρήση για modding και έχει περισσότερες ρυθμίσεις.
Μπορεί να αντικαταστήσει με ευκολία το BTTV καθώς παρέχει όλες τις λειτουργίες του πρώτου.
Αξίζει να αναφερθεί μιας και το BTTV είναι πιο γνωστό από το FFZ το εξής.
Το FFZ περιλαμβάνει όλα τα emotes/gif του BTTV κάτι που δεν κάνει το BTTV μη αντικαταστήσιμο για αυτό τον λόγο.
Μία απο τις βασικές λειτουγίες του είναι η δύνατοτητα για Custom Mod Actions/Icons.
Από τις ρυθμίσεις του Chat>Actions>In-Line μπορεί αρκετά εύκολα κάποιος να φτιάξει ανάλογα τις ανάγκες του τα δικά του Mod Icons
Στην παρακάτω φωτογραφία έχω φτιάξει button που κάνει timeout 2 λεπτά. Μπορεί κάποιος να προσθέσει όσα buttons χρειάζεται.
Στη συγκεκριμένη ρύθμιση απλά επιλέγουμε + New...>timeout user>και αλλάζουμε μόνο το duration.







Στη συνέχεια έχω φτιάξει ένα permit button επιλέγοντας + New...>chat command>άλλαξα το Lebel για να ξεχωρίζει σαν permit button
και συμπλήρωσα το εξής στο πεδίο Chat command !permit {{user.login}}









Στη συνέχεια έφτιαξα με τον ίδιο τρόπο και ένα button για shortcut mention
αυτή τη φορά προσέχουμε να ενεργοποιήσουμε την επιλογή κάτω κάτω ''paste this message..''







Αν για κάποιο λόγο οι επιλογές που θέλουμε να έχουμε σε buttons είναι πολλές για να εμφανίζονται όλες μαζί στο chat
μπορούμε να φτιάξουμε ένα contex menu έτσι ώστε να τις έχουμε σαν back up plan.
Η συγκεκριμένη ρύθμιση βρίσκεται λίγο πιο κάτω Chat>Actions>User Context
Φτιάχνουμε buttons με τον ίδιο τρόπο, εκεί μπορούμε να έχουμε είτε τα default του Twitch είτε τα custom που έχουμε φτιάξει.
Το menu ανοίγει στο chat κάνοντας δεξί κλικ σε ένα username







Στο παρακάτω gif φαίνεται τόσο η συγκεκριμένη λειτουργία στην πράξη όσο και το shortcut mention που αναφέρθηκε πιο πριν







Άλλη μία σημαντική λειτουργία για να αποφευχθούν λάθη από miss clicks
την στιγμή κάποιου mod action είναι η προσωρινή παύση του chat την στιγμή που το ποντίκι κάνει hover πάνω σε αυτό.
Η συγκεκριμένη βρίσκεται στο Chat>Actions>Behavior>Scrolling>Pause Chat Scrolling δεν χρειάζεται να αλλάξουμε κάτι άλλο σε αυτό.
Υπάρχει πιθανότητα να φανεί περίεργο στην αρχή για κάποιον αλλά με τον καιρό γίνεται συνήθεια.







Οι ρυθμίσεις του FFZ είναι πάρα πολλές και αξίζει να τις ψάξει κάποιος μία προς μία.
Ένα άλλο κομμάτι στο chat είναι η εμφάνιση. Είναι σημαντικό να φαίνεται το chat ξεκούραστο στο μάτι.
Υπάρχουν πάρα πολλές επιλογές σε αυτό το κομμάτι και το FFZ δίνει την ανάλογη προσαρμοστικότητα.


Αυτά ήταν τα βασικά εργαλεία από τα δύο πιο γνωστά extension του Twitch και τα οποία ο καθένας μπορεί να έχει εύκολη και δωρεάν πρόσβαση.
Αξίζει επίσης να στηρίζουμε αυτές τις προσπάθειες που κάνουν οι developers αυτών των λειτουργιών.
Τα Mod tools δεν σταματάνε όμως εδώ, συνεχίζουν σε ακόμα πιο εξειδικευμένο κομμάτι με τα Verified Bots,
τα IRC clients και μεγάλα κεφάλαια λειτουργιών όπως είναι οι Highlight Lists και τα Regular Expressions για πιο αυτοματοποιημένο modding.
Σε αυτά ίσως αναφερθώ σε κάποιο άλλο thread στο μέλλον.
